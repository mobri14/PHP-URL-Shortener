# PHP URL Shortener

## Overview

A lightweight URL shortening application built with PHP.

Users can enter a long URL and receive a short unique code.

## Features

* Generate short links
* Simple PHP implementation
* Beginner-friendly
* Lightweight

## Technologies

* PHP
* HTML

## Run

```bash
php -S localhost:8000
```

Open:

```text
http://localhost:8000
```
